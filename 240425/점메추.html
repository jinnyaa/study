랜덤 점메추 프로그램 작성

prompt창으로 '한식, 중식, 일식, 양식' 중 하나를 입력 하였을때 아래의 각 점심 메뉴가 나오게 코드를 작성하시오

  • 보기의 음식종류만 입력 가능할 것
  • if문과 Math.random() 활용할 것
  • Hint: 배열에서 랜덤한 값을 선택할시 ".length"를 사용
    .length(): 텍스트의 길이를 제한하기 위해서 사용하거나,String이나 Object List 배열의 길이를 가져와 for문으로 배열의 길이만큼의 index로 값을 가져올 때 사용
    ex.Math.random()*한식.length;
  • 아래 메뉴 참고 한식:김치찌개, 청국장,된장찌개,부대찌개,낙곱새,닭갈비
    일식:초밥,돈까스,가츠동,카츠나베,모밀소바
    중식:짜장면,짬뽕,마파두부,중국냉면,유린기
    양식:토마호크,파스타,감자뇨끼,피자,햄버거
